###  Projects

**[ Portfolio RAG](https://github.com/varunkumaruppula/portfolio-rag)**
A RAG system that answers questions about my own projects, grounded in my
actual write-ups via ChromaDB + a local Ollama LLM. Documents four real,
reproducible bugs found and fixed — an embedding-distance-scale mismatch, a
cross-document hallucination, an LLM faithfulness failure, and a
precision/recall trade-off in retrieval — each one traced to root cause
with a verified before/after fix.

**[ A/B Testing: Power Analysis & Peeking Bias](https://github.com/varunkumaruppula/ab-testing-experiment)**
A complete experimentation pipeline that doesn't stop at a clean t-test:
sizes a test with power analysis, then *proves* the cost of checking
significance too early by simulating 2,000 null-effect tests — naive daily
peeking inflates the false-positive rate to 25.9% against a nominal 5%.
Closes with an honest result: a significant lift overall, driven by one
segment, not independently significant in another.

**[ NYC Taxi Anomaly Detection Engine](https://github.com/varunkumaruppula/nyc-taxi-anomaly-detection)**
Unsupervised anomaly detection on 2.83M real NYC taxi trips — Isolation
Forest, SHAP explainability, PSI drift monitoring, a 7-tab Streamlit
dashboard with a local-LLM query layer — built entirely on an 8GB RAM
laptop with zero paid APIs. Eight documented bugs found and fixed,
including a 45x model bias traced to a single feature and a false drift
alarm traced to a calendar-windowing artifact.

** Human Disease Prediction from Tongue Images** *(solo capstone — repo coming soon)*
Two transfer-learning CNNs — ResNet50 (PyTorch) for diabetes, MobileNetV2
(TensorFlow/Keras) for oral cancer — deployed behind a Flask app with auth
and a SQLite-backed prediction history.

** Real-Time Social Media Sentiment Pipeline** *(AWS — repo coming soon)*
Serverless sentiment analysis on Kinesis, Lambda, and Comprehend, built to
replace a 24-hour batch ETL process with sub-minute latency.

** Comparative Protest-Pattern Analysis: U.S. vs. India** *(repo coming soon)*
Time-series, network, and geospatial analysis of ACLED conflict-event data
across two democracies — actor-network centrality, escalation-path
modeling, DBSCAN geo-clustering.
